3.9实验室日常小结

发表于 2019-03-09

就记一下今天干了些什么，不然老觉得自己一天没干什么，操作系统是Ubuntu16.04 LTS

conda环境管理

　　今天在服务器上尝试跑了一下github上faster R-CNN的tensorflow版本，用到的是python2.7，就自己用conda创了一个环境，用到的命令有：
创建环境：conda create -n py2.7 python=2.7
激活环境：conda activate py2.7
退出环境：conda deactivate
删除环境：conda remove -n py2.7 --all

Linux间传输文件

今天查了下两台Linux系统间如何传输文件，用的是scp，ssh也可以，但是我没试~
scp VGGnet_fastrcnn_iter_70000.ckpt hostname@ip:~/ Workspace/smt_models/
把本机的VGGnet_fastrcnn_iter_70000.ckpt文件传给远程服务器的~/Workspace/smt_models/目录下.

Linux系统间远程显示图片

ssh -X（超级好用我的妈！！）
因为我是在自己主机上用ssh连的服务器，两台都是Ubuntu的系统，服务器上用matplotlib画图显示不出来（具体原因我h还母鸡），所以网上查了一下，在用ssh时加上-X参数，那服务器上该显示的图片就会在我的主机上显示出来了.
